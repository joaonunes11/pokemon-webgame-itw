# Pokémon Web Game (ITW Project)

This is a complete interactive web game inspired by the Pokémon universe. It was developed as a final project for the subject "Introdução às Tecnologias Web (ITW)" and includes mini-games, interactive character selection, battles, sound, and video integration.

## 🎮 Project Highlights

- Interactive battle interface
- Dice game as a side mini-game
- Character and Pokémon selection
- Embedded video tutorial and background music
- Dynamic JavaScript-driven behavior

## 📂 Project Structure

```
pokemon-webgame-itw/
```

## 🚀 How to Launch

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Explore pages via navigation bar or links.
4. Enable audio/video permissions in your browser for full experience.

## 🛠 Technologies Used

- HTML5 & CSS3
- JavaScript (vanilla)
- Audio & Video API
- Responsive layout and event-based interactions
